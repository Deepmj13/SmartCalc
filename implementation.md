# 🚀 SmartCalc – All-in-One Utility Calculator App

SmartCalc is a modern **multi-utility calculator application for Android** built using Flutter.
It combines multiple everyday calculation tools into a single, clean, and high-performance app — similar to advanced calculator suites found on premium devices.

---

## 📱 Features

### 🧠 Core Calculators

- 📊 BMI Calculator
- 🎂 Age Calculator
- 📐 Area & Geometry Calculator
- 📏 Distance Calculator

---

### 💰 Finance Tools

- 💳 EMI Calculator
- 🧾 GST Calculator
- 📈 Interest Calculator (Simple & Compound)
- 💸 Discount & Profit/Loss Calculator

---

### 🔄 Conversion Tools

- 📏 Unit Converter (Length, Weight, Temperature, etc.)
- 💱 Currency Converter _(API-based)_
- 💾 Data Storage Converter

---

### ⏱ Time & Utility

- ⏳ Date Difference Calculator
- 🕒 Time Calculator
- 🌍 Time Zone Converter

---

### 🚗 Travel Tools

- ⛽ Fuel Cost Calculator
- 📍 Distance via Maps _(optional API)_
- 🚗 Mileage Calculator

---

### 🧑‍💻 Developer Tools _(Planned)_

- 🔢 Number Base Converter
- 🎨 Color Code Converter
- 📄 JSON Formatter

---

## 🏗 Architecture

This project follows:

- **Clean Architecture**
- **Feature-first structure**
- **MVVM pattern with Riverpod**

```
lib/
 ├── core/
 ├── features/
 ├── shared/
 └── main.dart
```

Each feature is modular and independently scalable.

## 📦 Installation

### Prerequisites

- Flutter SDK installed
- Android Studio / VS Code

### Steps

```bash
git clone https://github.com/your-username/smartcalc.git
cd smartcalc
flutter pub get
flutter run
```
